# Changelog

Формат: [SemVer](https://semver.org/lang/ru/), версии в порядке убывания.

## [1.2.0] - 2026-03-30

### Added
- Текущая стабильная версия модуля по `install/version.php`.
- Клиент-серверный калькулятор кастомных параметров SKU.
- Серверный endpoint расчёта цены и интеграция с корзиной.
- Инжект `window.pmodConfig` и frontend lifecycle для UI-калькулятора.

---

Рекомендация по ведению:
- повышать `MAJOR` при несовместимых изменениях API/контракта данных;
- повышать `MINOR` при добавлении функциональности без breaking changes;
- повышать `PATCH` при исправлениях без изменения контракта.
